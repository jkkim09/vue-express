# 환경
express(nodejs) + vuejs (webpack) 으로 Web 환경을 구성하였다. <br>

# 설치

## Front-end(vuejs) 설치

`````bash
$ cd ./frontend
`````

`````bash
$ npm install
`````
## Front-end(vuejs) 실행

`````bash
$ cd ./frontend
`````

`````bash
$ npm run start
`````
## Front-end(vuejs) Build

`````bash
$ cd ./frontend
`````

`````bash
$ npm run build
`````

## Project 배포

`````bash
$ docker run -d -p {외부포트}:{내부포트} {tagname:version}
`````
